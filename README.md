🦈 Shark Attack Analysis (EDA Project)
📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a dataset of shark attacks recorded over the past 100 years. The goal is to uncover patterns, trends, and risk factors associated with shark attacks using Python.

The analysis includes data cleaning, statistical summaries, and visualizations to better understand how factors like age, gender, location, activity, and shark species influence attack occurrences.

🎯 Objectives
Perform data cleaning and preprocessing
Conduct univariate, bivariate, and multivariate analysis
Use groupby, pivot tables, and correlation analysis
Generate insightful visualizations
Identify key trends, patterns, and anomalies

🛠️ Tools & Libraries Used
Python
Pandas – Data manipulation
NumPy – Numerical operations
Matplotlib – Data visualization
Seaborn – Statistical visualization

📂 Dataset Description

The dataset contains information about shark attacks, including:

Year of attack
Country / Location
Victim details (Age, Gender)
Activity (Swimming, Surfing, etc.)
Shark species
Fatality (Yes/No)


🔍 Key Insights
Males are more frequently attacked than females
Young adults (15–35) are the most affected age group
Surfing and swimming are high-risk activities
Certain countries report higher attack frequencies
A few shark species dominate attack cases
Weak correlation between numerical variables suggests external factors influence attacks more

⚠️ Limitations
Missing or incomplete data in some fields
Limited environmental variables (e.g., water conditions)
Possible reporting bias across regions
